Gryn

Norwegian: Gryn, et regnskapsprogram for GNU/Linux

English: Gryn, an accounting program for GNU/Linux systems

To check out the last revision of the source tree you need a svn client:

svn checkout svn://svn.berlios.de/gryn
Then read the README file in the trunk directory. Go to http://gryn.berlios.de to download the newest release.
